## 项目介绍
基于springboot的汽车租赁管理系统：前端 html、jquery，后端 maven、springmvc、spring、mybatis；角色分为管理员、用户；集成自驾租车、企业服务、汽车信息管理、发布租车、订单查询等功能于一体的系统。
